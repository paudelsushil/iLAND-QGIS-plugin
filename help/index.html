iLAND Workbench Help

This local help page is bundled with the QGIS plugin package.

Quick Start

  1. Open the plugin from toolbar or menu.
  2. Set a valid iLAND repository root containing src/.
  3. Use Runtime tab to check/install/activate runtime as needed.
  4. Use Workflow tab to build or run commands.
